# Simple 2D Platformer

A very simple 2D platformer game built with HTML5 Canvas and vanilla JavaScript.

## Features

- Player character with smooth movement
- Jump mechanics with gravity
- Multiple platforms to navigate
- Collectible coins for scoring
- Game over and restart functionality
- Responsive controls

## How to Play

1. Open `index.html` in a web browser
2. Use the following controls:
   - **Arrow Left/Right**: Move the player left or right
   - **Space or Arrow Up**: Jump
   - **R**: Restart the game

## Game Objective

- Navigate through the platforms
- Collect all the golden coins to increase your score
- Don't fall off the screen!
- Each coin is worth 10 points

## Files

- `index.html` - Main HTML file with canvas element
- `game.js` - Game logic and mechanics
- `style.css` - Styling for the game interface

## Technical Details

- Uses HTML5 Canvas for rendering
- Implements basic physics (gravity, collision detection)
- Pure vanilla JavaScript with no dependencies
- Runs entirely in the browser
